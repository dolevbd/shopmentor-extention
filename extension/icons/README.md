# Icons Folder

Place your extension icons here.

## Required Icon Sizes

For Chrome Web Store, you need icons in the following sizes:

- **16x16** - Toolbar icon (small)
- **48x48** - Extension management page
- **128x128** - Chrome Web Store listing and extension details page

## File Names

Name your icon files:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

Or use a single name pattern:
- `icon-16.png`
- `icon-48.png`
- `icon-128.png`

## Format

- **Format**: PNG (recommended) or JPEG
- **Background**: Transparent or solid color
- **Design**: Should represent ShopMentor AI (shopping assistant, price comparison, etc.)

## After Adding Icons

1. Update `manifest.json` to reference the icons
2. Test the extension to see icons in Chrome
3. Icons will appear in:
   - Extension toolbar
   - Extension management page (`chrome://extensions/`)
   - Chrome Web Store listing
